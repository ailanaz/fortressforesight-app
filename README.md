# FortressForesight App

Protect what protects you.

FortressForesight is a home readiness, records, and recovery support app for
homebuyers and homeowners.

## Tech Stack

- React with Vite
- React Router
- Firebase-ready environment setup
- GitHub Pages deployment via GitHub Actions

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in the values you want to use locally.

Never commit `.env`.

## Deployment

- Repo: `ailanaz/fortressforesight-app`
- Custom domain: `fortressforesight.app`
- Hosting: GitHub Pages via GitHub Actions
